## Notes

- All addresses are valid US addresses
- ZIP codes match the city/state combinations
- Phone numbers are in US format but are fictional
- Street 2 (secondary address) is optional and can be left empty
- All addresses use "US" as the country code

## EasyPost Test Mode

When using EasyPost in test mode, you can use any valid US address format. The API will accept these addresses and generate test labels without charging your account.
